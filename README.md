# Angular Electron Starter

Electron, Angular & TypeScript Starter for building awesome desktop applications.

### Start

```sh
git clone https://github.com/jkuri/angular-electron-starter.git --depth 1
npm install
```

### Development Mode

This enables livereload.

```sh
npm start
```

### Production Build

```sh
npm run build
```

### Wrap as Application

```sh
npm run dist
```
